GROMMET WEBSITE REDESIGN

Desktop Version

Mobile Version


Overview

I was given a task on redesigning Grommet's current website for a desktop and mobile version. I followed a mobile first approach, and faced many challenges in creating an e-commerce website for such a unique company that sells all sorts of items.
GOALS: My goals for the designs were to have a clean UI that was easy to navigate. I wanted to avoid clutter, but still not have anything hidden behind a hamburger menu to grab the user's attention and keep them engaged to click around the different categories and explore more products.



Ideation


I was given a wireframe and three website examples of what Grommet was looking for. I started sketching some ideas.


Mockup

Using the original website for information, I went through and circled all the super important information for the mobile version on the very right.
MOBILE: I focused on mobile first by going through the wireframe and website provided to find the most important informations to include. I decided to have the details be collapsible to keep the page clean and not overwhelm the user with too much information. They can click and the section will expand.
WEBSITE: I wanted an easier way for the users to navigate quickly around the desktop version so I opted for quick links. These are the 3 roll overs below the 'Add to Wishlist' button for the user to click on whichever section they're interested in seeing whether it's product details, reviews, or about the company. This will automatically jump them to that section.
