Overview

I redesigned this website for my interactive design class. I created wireframes, mockups, and developed a responsive website based on the mockups using bootstrap, Javascript, html 5, and CSS.
PROBLEM: There are alot of items in the drop down menus and the wording is very long. The information in each of the category are often repeated in another page. The information is jumbled and hard to find.
GOALS: I wanted to simplify the website and make a clean UI that wasn't so cluttered compared to the original. This will help the users find their information a lot faster whether their businesses or regular people wanting to volunteer and get involved.



Takeaway


This project taught me how to take a design of a mockup and turn it into a responsive website using Bootstrap, Html 5, and CSS. It also provided a learning experience to try different features using Javascript.
